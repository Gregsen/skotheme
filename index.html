Welcome to SkoTheMe.

SkoTheMe is a tool to support your daily thesauri merging processes. This script is the result of my bachelor thesis written at the University of Applied Sciences Potsdam

SkoTheMe provides several methods to work with skos thesauri. Basically, SkoTheMe reads your SKOS files, scrapes the labels and uses several methods to compare them. The result is a new SKOS file, containing your input files as well as new relations between the concepts. A full documentation can be found here.

Requirements

SkoTheMe requires Python 2.7 (not tested with Python 3). Additionally, it uses (of course!) RDFLib and NLTK.
